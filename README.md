# Controle de Prestação de Serviços

Sistema web para controle de pagamentos de prestação de serviço. Funciona direto no navegador, sem servidor ou banco de dados.

## Funcionalidades

- Cadastro de lançamentos com fornecedor, serviço, valor, período e vencimento
- Controle de status: pendente, pago, vencido e pago parcialmente
- Filtros por fornecedor, status e período
- Relatório com totais de valor lançado, pago e saldo em aberto
- Impressão do relatório
- Dados salvos no navegador (localStorage)
- Suporte a modo escuro

## Como usar

Abra o arquivo `index.html` no navegador — nenhuma instalação necessária.

Ou acesse via GitHub Pages: `https://<seu-usuario>.github.io/<nome-do-repo>/`

## Publicar no GitHub Pages

Após subir o repositório, vá em **Settings → Pages → Source → Deploy from a branch → main / root** e salve.
